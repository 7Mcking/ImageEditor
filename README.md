## Image Viewer using Qt

A basic application to load, save images and perform basic image processing steps.

### Dependencies
QpenCV
QT5

### Building and running
```git clone https://github.com/7Mcking/ImageEditor.git```

``` mkdir build && cd build```

```cmake ..```

```make```

```./ImageViewer```
